# TOOL-0002: Media Pool Metadata Tool

## Tool
- `anica.media_pool/list_metadata`
- `anica.media_pool/remove_by_id`
- `anica.media_pool/clear_all`

## Purpose
Get runtime media facts from current project media pool, and remove pool entries without touching physical files.

## Typical Use
- Answer questions about codec/fps/resolution/duration.
- Compare media items.
- Build export recommendations based on actual source media.

## Common Call
`anica.media_pool/list_metadata`
```json
{
  "include_missing_files": true,
  "include_file_stats": true,
  "include_media_probe": true
}
```
`anica.media_pool/remove_by_id`
```json
{
  "id": "/absolute/path/to/clip.mp4"
}
```
`anica.media_pool/clear_all`
```json
{}
```

## Key Fields Returned
- `name`, `path`, `media_kind`
- `width`, `height`, `fps`
- `video_codec`, `audio_codec`
- `duration_seconds`

## Guidance
- Do not guess media facts from docs/source code.
- Always treat this tool as runtime source of truth for media attributes.
- `remove_by_id` and `clear_all` remove project media-pool references only.
- These tools must never delete filesystem media files.
