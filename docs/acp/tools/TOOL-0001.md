# TOOL-0001: Docs Tools (`anica.docs/*`)

## Tools
- `anica.docs/list_files`
- `anica.docs/read_file`

## Purpose
Read ACP documentation files so the agent can answer with documented facts.

## Common Calls
```json
{"subdir":"acp/export"}
```
```json
{"path":"acp/export/catalog.md","max_chars":60000}
```

## Response Notes
- `list_files` returns file paths relative to docs root.
- `read_file` returns `exists`, `content`, and optional `error`.

## Failure Cases
- `docs subdir does not exist`
- `invalid docs subdir path`
- `file not found`
- `invalid docs file path`

## Guidance
- Use catalog-first (`acp/*/catalog.md`) before opening topic docs.
- If file missing, apologize briefly and give best-effort inferred answer.
