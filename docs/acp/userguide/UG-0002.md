# UG-0002: Build Anica And Connect ACP

## Requirements

- Rust toolchain from `rust-toolchain.toml`
- FFmpeg and FFprobe

## macOS

```bash
brew install git ffmpeg
cargo run
```

## Linux

```bash
sudo apt install ffmpeg
cargo run
```

## Windows

```powershell
powershell -ExecutionPolicy Bypass -File .\scripts\setup_media_tools.ps1 -Yes
cargo run
```

## Verify

```bash
ffmpeg -version
ffprobe -version
```

## ACP Helper

The cargo runner rebuilds `anica-acp` automatically when the helper is missing or stale.
