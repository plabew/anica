# Third-Party Notices

Anica source code is Apache-2.0 unless a file states otherwise. Runtime tools keep their upstream licenses.

## FFmpeg/FFprobe

Anica uses FFmpeg/FFprobe for media preview, export, proxies, thumbnails, waveform preparation, and media analysis. If you redistribute FFmpeg binaries with Anica, include the exact upstream license notices and source-obtain information for the build shipped.

Recommended redistributable builds should be LGPL-only, for example with GPL/nonfree components disabled.

Reference: https://ffmpeg.org/legal.html
