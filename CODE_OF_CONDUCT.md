# Code of Conduct

## Our Commitment

We are committed to providing a friendly, safe, and welcoming environment for
everyone, regardless of background, identity, experience level, or personal
characteristics.

## Expected Behavior

- Be respectful and constructive in discussions and reviews.
- Assume good intent and ask for clarification before escalating.
- Focus feedback on code, behavior, and outcomes, not on people.
- Accept accountability and correct mistakes quickly.

## Unacceptable Behavior

- Harassment, discrimination, hate speech, or personal attacks.
- Threats, intimidation, or doxxing.
- Trolling, bad-faith provocation, or repeated disruption.
- Publishing private information without explicit permission.

## Scope

This Code of Conduct applies to all project spaces, including:

- GitHub issues, pull requests, and discussions
- community chats and calls related to this project
- any public or private interaction where a person is representing the project

## Reporting

If you experience or witness unacceptable behavior, report it privately to the
maintainers through repository contact channels. Please include:

- what happened
- links/screenshots/logs if available
- when and where it happened
- any immediate safety concerns

## Enforcement

Maintainers may take any action they deem appropriate, including warning,
temporary restrictions, or removal from project spaces. Reports will be handled
promptly and as confidentially as practical.

## Project Note

By participating in this project, you agree to follow this Code of Conduct.
